# YSB Game Editor

YSB Game Editor is a desktop tool for managing, translating, and reviewing RPG Maker game text data.

## Features

- RPG Maker project loading
- Map text preview
- Batch translation workflow
- Unified translation replacement
- Database text editing
- Project-based save/cache system
